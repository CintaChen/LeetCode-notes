\# LeetCode 42. Trapping Rain Water (接雨水) ## 📝 题目描述 给定 `n` 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。 **示例：** 输入：height = [0,1,0,2,1,0,1,3,2,1,2,1] 输出：6 ## 💡 解题思路与想法 ### 1. 初始想法（暴力/削层法） 最开始想到的是“一层一层削”的方法，即从最低处开始一层层遍历。虽然可行，但时间复杂度极高，在大数据量下会超时。 ### 2. 双指针法（最终方案） 使用双指针从两端向中间逼近，同时维护两个变量 `left_max` 和 `right_max` 分别记录左右两边遇到的最高柱子。 **核心逻辑：** *   水的容量取决于**短板**。 *   比较 `left_max` 和 `right_max`：    *   如果 `left_max < right_max`：说明左边的短板决定了当前位置的水位上限。此时处理 `left` 指针，如果当前高度小于 `left_max`，则累加雨水量；否则更新 `left_max`。    *   反之亦然，处理 `right` 指针。 **为什么不会漏解？** 因为在移动短板一侧时，另一侧一定存在一个不低于当前短板的高度作为边界，所以当前位置的储水量是可以被准确计算的。
